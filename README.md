# Number-Guessing-Game
This is a simple Number Guessing Game developed in Python. The game randomly selects a number, and the player has to guess it. The program provides hints whether the guess is too high or too low until the player guesses correctly.

Features:

Random number generation

User input handling

Feedback on guesses (“Too high” / “Too low”)

Tracks the number of attempts

Technologies Used:

Python 3.x

VS Code (for development)

Git & GitHub (for version control)

How to Play:

Run the program (python number_guessing_game.py).

Enter your guesses in the terminal.

Keep guessing until you find the correct number.

The game will display the number of attempts taken.
